# Artificial Intelligence in Cybersecurity Defense

This repository contains my graduate research paper on the role of artificial intelligence in modern cybersecurity defense.

## Overview
This research explores how AI improves:
- Threat detection
- Malware analysis
- Phishing detection
- Intrusion detection and threat intelligence

It also examines the challenges of AI in cybersecurity, including:
- Adversarial attacks
- Data quality issues
- Ethical and governance concerns
